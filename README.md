# Retail-Customer-Transaction
El objetivo es integrar y analizar las diversas fuentes de datos de una operación de Retail para entender cómo reacciona el consumidor y qué tan rentables son los esfuerzos de marketing.  Se transformaron datos dispersos de transacciones, perfiles de clientes, campañas publicitarias y reseñas de productos orientada a la toma de decisiones.
